# 💳 Razorpay Clone

A pixel-perfect, fully responsive clone of the [Razorpay](https://razorpay.com) landing page — built with **HTML**, **Tailwind CSS**, and **Feather Icons**.

## ✨ Features

- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop using Tailwind breakpoints
- 🎨 **Pixel-perfect UI** — Closely replicates the original Razorpay landing page design
- 🧭 **Sticky Navbar** — With hover underline animations and responsive collapse
- 💳 **Custom SVG Cards** — Feature cards with unique clip-path polygon shapes
- 🌀 **CSS Animations** — Smooth infinite company logo scroll using `@keyframes`
- 🌗 **Dark & Light Sections** — Business Banking section with deep dark theme contrast
- 🖱️ **Hover Effects** — Scale, color, and background transitions throughout
- 🦶 **Detailed Footer** — Multi-column layout with newsletter input, social links, and certificate badges

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/your-username/razorpay-clone.git
cd razorpay-clone
```

### 2. Open in browser
```bash
# macOS
open index.html

# Windows
start index.html
```

> No build tools, no npm, no setup — just open and run!

## 📱 Responsive Breakpoints

| Breakpoint | Width | Layout |
|---|---|---|
| Mobile (default) | < 768px | Single column, hamburger nav |
| Tablet `md` | ≥ 768px | Two-column grids, images visible |
| Desktop `lg` | ≥ 1024px | Full navbar, three-column grids |
